# Observation Vector Datasets

- Stock data: 
    - raw: jul/, from http://cs.binghamton.edu/~xianli/truthfinding.htm
    - formatted into stock.dat and stock_truth.dat, using `python preprocess_stock.py`
- Synthetic data: generated using `python synthetic.py 0.7 0.25`, `python synthetic.py 0.7 0.15`, and `python synthetic.py 0.8 0.25`
